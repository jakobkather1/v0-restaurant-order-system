# Notification Sound Setup

To enable audio notifications for new orders, add a notification sound file to this directory.

## Required file:
- `notification.mp3` - A short, pleasant notification sound (1-2 seconds)

## Recommended sources for free notification sounds:
- https://freesound.org/ (CC0 or CC-BY licensed sounds)
- https://mixkit.co/free-sound-effects/notification/
- https://pixabay.com/sound-effects/search/notification/

## Example search terms:
- "bell notification"
- "order alert"
- "chime"
- "ding"

## File requirements:
- Format: MP3
- Duration: 1-2 seconds
- Size: < 100KB
- Volume: Pre-normalized to comfortable listening level

Once added, the system will automatically play this sound when new orders arrive in the admin panel.
